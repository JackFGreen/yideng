### 3. 同源访问限制
1. cookie，localStorage(5M) 和 IndexDB 无法读取
2. DOM 不能获取
3. ajax 不能发送

### 4. 设置同源策略
共享 cookie  
一级域名不同，二级域名相同

```js
document.domain
```

```
test01.xxx.com/a.html
  document.domain = 'xxx.com'
  document.cookie = 'test=01'

test02.xxx.com/b.html
  document.cookie
```
